Desarrollo y proposito del lenguaje HMTL5

HTML5 es la quinta versión del lenguaje estándar de marcado utilizado para crear páginas web. Fue desarrollado por el W3C (World Wide Web Consortium) y el WHATWG (Web Hypertext Application Technology Working Group), como una evolución de HTML4, con el objetivo de modernizar la web.
Desarrollo Comenzó en 2004 y se convirtió en una recomendación oficial en 2014. Nació para resolver las limitaciones de HTML anterior y responder a las nuevas necesidades del desarrollo web moderno. Integra directamente funciones que antes requerían plugins externos como Flash (por ejemplo: audio, video y gráficos).
